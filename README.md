# AzuraCast Home Assistant Add-on Repository

This repository contains the AzuraCast add-on for Home Assistant.

## Installation

1. In Home Assistant, navigate to **Settings** → **Add-ons** → **Add-on Store**
2. Click the three dots in the top right corner and select **Repositories**
3. Add this repository URL: `https://github.com/roulianosss/azuracast-addon-ha`
4. Find "AzuraCast" in the add-on store and click **Install**
