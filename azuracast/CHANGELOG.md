# Changelog

All notable changes to this add-on will be documented in this file.

## [1.0.0] - 2025-12-16

### Added
- Initial release of AzuraCast Home Assistant add-on
- Full AzuraCast web radio management suite integration
- Support for multiple radio stations
- Automatic MySQL password generation
- Persistent data storage
- Docker-in-Docker implementation
- Health check monitoring
- Comprehensive documentation
- Multi-architecture support (amd64, aarch64, armv7)
- Port mapping for web interface and radio streams
- SSL/TLS support with Let's Encrypt
- RESTful API access
- Configuration options for customization

### Features
- Web-based radio station management
- AutoDJ with playlist management
- Live broadcasting capabilities
- Listener statistics and analytics
- Media file manager
- Scheduled playlists
- Request system for listeners
- SFTP access for file uploads
- Multiple streaming formats (Icecast, SHOUTcast)

### Configuration
- Configurable HTTP/HTTPS ports
- MySQL database customization
- Let's Encrypt SSL support
- Persistent storage mapping

### Documentation
- Complete installation guide
- Configuration examples
- API usage documentation
- Troubleshooting guide
- Home Assistant integration examples
